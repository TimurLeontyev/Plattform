# Increvio Homepage

Single-file static homepage for Increvio.

## Export

The complete page is contained in `index.html`.

- CSS is embedded inline in `<style>`
- Project JavaScript is embedded inline in `<script>`
- No local CSS, JavaScript, image, or framework dependencies
- External dependencies are limited to Google Fonts and the allowed GSAP CDN scripts

Open `index.html` directly in a browser, or deploy it to any static host such as Netlify.
